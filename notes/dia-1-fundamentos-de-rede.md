# Dia 1 — Fundamentos de Rede

Hoje estudei conceitos básicos sobre redes e internet.

## IP (Internet Protocol)

O IP é um identificador único atribuído a cada dispositivo em uma rede.  
Ele permite que dispositivos se comuniquem entre si através da internet.

### IP observado
172.21.5.2

---

## DNS (Domain Name System)

O DNS é responsável por traduzir nomes de domínio em endereços IP.

Exemplo:

www.google.com → endereço IP

Isso permite que usuários acessem sites sem precisar decorar números.

---

## HTTP e HTTPS

HTTP é um protocolo utilizado para transferência de dados na web.

Ele permite comunicação entre:

- Cliente (navegador)
- Servidor (site)

### HTTPS

HTTPS é a versão segura do HTTP.

Ele utiliza criptografia para proteger os dados transmitidos.

---

## TCP

TCP é um protocolo que garante que os dados sejam enviados corretamente entre dispositivos.

Ele garante:

- Entrega confiável
- Ordem correta dos pacotes
- Reenvio de dados perdidos
