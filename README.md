📚 Conceitos aprendidos
🌐 IP (Internet Protocol)

O IP (Internet Protocol) é um identificador único atribuído a cada dispositivo em uma rede. Ele permite que dispositivos se comuniquem entre si na internet.

🔍 Endereço IP observado
172.21.5.2

Esse endereço pertence à Classe B, onde:

A primeira metade do IP representa o endereço de rede
A segunda metade representa o endereço de host
172.21 = Rede
5.2 =  Host

🔢 Classes de Endereço IP (IPv4)

Os endereços IP são divididos em classes, que ajudam a definir o tamanho da rede e a quantidade de dispositivos possíveis.

Classe	
A	1 – 126	= Redes muito grandes
B	128 – 191	= Redes médias
C	192 – 223	= Redes pequenas

🧠 Como cada classe é usada:

Classe A:
Usada em redes gigantes (ex: grandes organizações)
Poucas redes, mas com muitos dispositivos
Maior parte do IP é destinada aos hosts

Classe B:
Usada em redes de tamanho médio
Equilíbrio entre quantidade de redes e dispositivos
Exemplo: empresas, universidades

Classe C:
Usada em redes pequenas
Muitas redes, mas com poucos dispositivos em cada
Muito comum em redes locais (LAN), como casas e pequenos escritórios

🔍 Exemplo prático
IP analisado: 172.21.5.2
Pertence à Classe B
Parte do endereço identifica a rede
Parte identifica o dispositivo (host) dentro da rede
